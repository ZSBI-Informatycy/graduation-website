Information
This website was made as a graduation project. It's nothing fancy, just enough to prove my knowledge, experience and project management. The entire source code of the website is open and free to download via the Github repository on the Projects page. The website was being developed at a stable pace and with Github you can check the commits I made from 2 different computers and also see traffic, language use and the version control.

What was used for the project:

  • Microsoft Visual Studio Code: for writing the code and managing the server in the integrated terminal,
  • Python: for local web-hosting and debugging JavaScript,
  • Firefox Developer Edition: for previeving the webpage and using its developer functions during the development of the project,
  • Git Bash: for version control, project maintenance, management and more,
  • GitHub: for hosting the local repository on the internet,
  • Chromium Dev: for testing the website on a chrome-based browser,
  • StackOverflow and W3Schools: for CSS and HTML DOM reference, and occasional documentation
